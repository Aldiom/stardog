Class LaserBeam

source code

LaserBeam(game, laser, damage, range) -> new LaserBeam

A LaserBeam is the projectile of a Laser. They are line segments that reach their end point instantly. A LaserBeam has a different collision mechanism: they use line/circle collision, and it is checked during initialization.
