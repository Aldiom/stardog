Class Floater

source code

creates a floater with position (x,y) in pixels, speed (dx, dy) in pixels per second, direction dir where 0 is pointing right and 270 is pointing up, radius radius (for collision testing), and with the image image. Image should be a string of a file name without an axtension- there should be both a .gif and a .bmp, which is used depends on the pygame support on the run system.
