Class TopLevelPanel

source code

TopLevelPanel(rect) -> TopLevelPanel. Like a panel, but has a handleEvent() method and draws subpanels to a buffer. Must be the ultimate parent of any panel that responds to events, and events should all be passed to .handleEvent(event).
